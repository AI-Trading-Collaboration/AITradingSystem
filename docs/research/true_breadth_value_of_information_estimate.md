# True Breadth Value of Information Estimate

YAML：`inputs/research_reviews/true_breadth_value_of_information_estimate.yaml`

## Participation proxy 已经证明了什么

Participation proxy 证明 breadth / participation 方向值得继续看，但当前 proxy 只基于
ETF ratios、listing-status 类辅助信号或低成本 holdings 入口，不是 true PIT breadth。
它不能证明 Nasdaq-100 每日真实成分集合，也不能避免 survivorship bias。

## True breadth 可能补足什么

合格 true breadth source 可以补足：

- historical constituents；
- delisted securities；
- daily membership snapshot；
- constituent-level price join；
- equal-weight vs cap-weight participation；
- concentration / contribution；
- sector / semiconductor participation。

这些正对应 post-2085 gate denial 后剩余的 first-layer blocker。

## 成本价值判断

如果 Norgate trial 能证明 daily membership、delisted coverage、Python access 和 local
cache 权限，信息增量可能显著高于 proxy，值得 owner review trial。若 true breadth trial
仍无法解决 2022 stress slice、beta/TQQQ dependency 或 false risk-off blocker，则应长期
暂停 first-layer 自动策略方向，只保留 diagnostic-only research。
