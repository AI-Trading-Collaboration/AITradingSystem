# High-Intensity Observe Continuation Scope

- continuation_mode: `observe_only`
- manual_review_context_allowed: `True`
- automatic_exposure_cap_allowed: `False`
- runtime_scheduler_enabled: `False`

Manual-review context 只能展示 warning / event / cluster / caveat context，不能解释为减仓、target weight、rebalance 或 broker order。
