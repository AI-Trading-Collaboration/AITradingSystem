# Norgate Paid Platinum Decision Gate

配置：`config/research/norgate_paid_platinum_decision_gate.yaml`

允许状态：

- `NORGATE_PAID_PLATINUM_RECOMMENDED`
- `NORGATE_PAID_PLATINUM_DEFERRED`
- `NORGATE_NOT_SUITABLE`
- `NORGATE_TRIAL_INCONCLUSIVE`

推荐正式 Platinum 仅当：

- Python access confirmed。
- Membership query confirmed。
- Delisted visibility confirmed or sufficiently documented。
- Snapshot prototype ready。
- 2Y breadth prototype generated。
- Cache / license governance passed。
- Expected research value > cost。

任何 purchase 都必须 owner manual approval。Gate 输出不能恢复 first-layer、v4、promotion、
paper-shadow、production 或 broker。
