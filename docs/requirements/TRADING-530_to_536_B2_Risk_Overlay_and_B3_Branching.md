## 必需输出

### TRADING-530

输出 `b2_risk_heavy_window_evaluation.json/md`，覆盖 rapid drawdown、slow drawdown、
high-volatility sideways、semiconductor correction、V-shaped recovery 和 false risk-off
cluster，至少包含：

- risk trigger count / dates；
- exposure scaler changes；
- drawdown / return / turnover / cost delta vs B0；
- false risk-off count；
- re-entry lag；
- V-shaped recovery opportunity cost。

状态必须为：

- `B2_RISK_OVERLAY_PROMISING`
- `B2_RISK_OVERLAY_MIXED`
- `B2_RISK_OVERLAY_WEAK`
- `B2_RISK_OVERLAY_NOT_TRIGGERED`
- `B2_RISK_OVERLAY_BINDING_ISSUE`

### TRADING-531

输出 `b3_slow_tilt_signal_direction_audit.json/md`，至少包含：

- asset-level tilt direction；
- wrong-tilt dates；
- relative strength signal lag；
- contribution by asset；
- overweighted underperformers；
- underweighted outperformers；
- turnover generated by tilt changes；
- signal normalization inverted / lagged / overreactive diagnosis。

状态必须为：

- `B3_SIGNAL_DIRECTION_INVERTED`
- `B3_SIGNAL_LAG_DOMINANT`
- `B3_SIGNAL_NOISY`
- `B3_ASSET_MAPPING_ISSUE`
- `B3_WINDOW_SPECIFIC_WEAKNESS`
- `B3_REDESIGN_REQUIRED`
- `B3_REJECT_CURRENT_FORM`

- `B1_OPTIONAL_WRAPPER`
- `B1_NEEDS_REDESIGN`
- `B1_REJECTED`
